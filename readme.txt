it is just an experiment in angualar2. 
No special design and no fully responsive, just flexibility (js oriented experiment).
Front part works in "category" menu, whre you can choose category and watch items which belong to them correspondingly.
You can set up category(multiple category if you want) via "admin" menu, by choosing products tab.
Project uses database emulator. Backend will be written with node.js.
 

"npm install" set up environment
"npm start" launches website
